TIPS FOR USING JAVATOR!

  • This calculator takes both input from clicking the buttons as well as from the keyboard.Give it a go!
  • You can use operators both from the numpad as well using the (shiftKey+desired_operator).For example shiftKey+8 results into 'X'(multiply) operator
  • This calculator can calculate only upto maximum safe integer of number type in javascript i.e 9007199254740991 or (2)^53-1.
  • Lastly enjoy using the calculator app, any short comings will be eradicated in the version 2 of the app with more functionality and bigger range
